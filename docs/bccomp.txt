语法:
   int bccomp    ( string $left_operand = &quot;&quot;   , string $right_operand = &quot;&quot;   [, int $scale = int  ] )
说明:
Comparestheleft_operandtotheright_operandandreturnstheresultasaninteger.
参数:
  参数              left_operand                  The left operand, as a string.                        right_operand                  The right operand, as a string.                        scale                  The optional scale parameter is used to set the       number of digits after the decimal place which will be used in the       comparison.                       
返回值:
  返回值     Returns 0 if the two operands are equal, 1 if the   left_operand is larger than the    right_operand, -1 otherwise.   
示例:

<?php
echo bccomp('1', '2') . "\n";   // -1
echo bccomp('1.00001', '1', 3); // 0
echo bccomp('1.00001', '1', 5); // 1
?>
