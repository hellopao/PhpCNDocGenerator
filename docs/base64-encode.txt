语法:
   string base64_encode    ( string $data   )
说明:
使用base64对data进行编码。
参数:
  参数              data                  要编码的数据。                     
返回值:
  返回值     编码后的字符串数据， 或者在失败时返回 FALSE。   
示例:

<?php
$str = 'This is an encoded string';
echo base64_encode($str);
?>
以上例程会输出：
VGhpcyBpcyBhbiBlbmNvZGVkIHN0cmluZw==
