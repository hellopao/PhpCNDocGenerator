语法:
      string decoct       ( int $number      )
说明:
返回一字符串，包含有给定number参数的八进制表示。所能转换的最大数值为十进制的4294967295，其结果为&quot;37777777777&quot;。
参数:
  参数              number                  待转换的十进制值                     
返回值:
  返回值     number     参数八进制表示的字符串。   
示例:

<?php
echo decoct(15) . "\n";
echo decoct(264);
?>
以上例程会输出：
17
410
