语法:
   string strtolower    ( string $string   )
说明:
将string中所有的字母字符转换为小写并返回。
参数:
  参数              string                  输入字符串。                     
返回值:
  返回值     返回转换后的小写字符串。   
示例:

<?php
$str = "Mary Had A Little Lamb and She LOVED It So";
$str = strtolower($str);
echo $str; // 打印 mary had a little lamb and she loved it so
?>
