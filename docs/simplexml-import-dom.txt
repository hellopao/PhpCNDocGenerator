语法:
   SimpleXMLElement simplexml_import_dom    ( DOMNode $node   [, string $class_name = &quot;SimpleXMLElement&quot;  ] )
说明:
ThisfunctiontakesanodeofaDOMdocumentandmakesitintoaSimpleXMLnode.ThisnewobjectcanthenbeusedasanativeSimpleXMLelement.
参数:
  参数              node                  A DOM Element node                        class_name                  You may use this optional parameter so that       simplexml_import_dom() will return an object of        the specified class. That class should extend the        SimpleXMLElement class.                     
返回值:
  返回值     Returns a SimpleXMLElement 或者在失败时返回 FALSE.   
示例:

<?php
$dom = new DOMDocument;
$dom->loadXML('<books><book><title>blah</title></book></books>');
if (!$dom) {
    echo 'Error while parsing the document';
    exit;
}
$s = simplexml_import_dom($dom);
echo $s->book[0]->title;
?>
以上例程会输出：
blah
