语法:
   string tempnam    ( string $dir   , string $prefix   )
说明:
在指定目录中建立一个具有唯一文件名的文件。如果该目录不存在，tempnam()会在系统临时目录中生成一个文件，并返回其文件名。
参数:
  参数              dir                  The directory where the temporary filename will be created.                        prefix                  产生临时文件的前缀。            Note:                Windows uses only the first three characters of prefix.                            
返回值:
  返回值     返回新的临时文件名，出错返回 FALSE。   
示例:

<?php
$tmpfname = tempnam("/tmp", "FOO");
$handle = fopen($tmpfname, "w");
fwrite($handle, "writing to tempfile");
fclose($handle);
// do here something
unlink($tmpfname);
?>
