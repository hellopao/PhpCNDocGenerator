语法:
   string XMLWriter::outputMemory    ([ bool $flush = true  ] )
说明:
过程化风格
参数:
  参数         xmlwriter仅用于过程调用。被修改的 XMLWriter resource。此资源来自于对 xmlwriter_open_uri() 或xmlwriter_open_memory() 的调用。         flush                  Whether to flush the output buffer or not. Default is TRUE.                     
返回值:
  返回值     Returns the current buffer as a string.
