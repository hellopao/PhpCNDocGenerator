语法:
   resource odbc_exec    ( resource $connection_id   , string $query_string   [, int $flags  ] )
说明:
SendsanSQLstatementtothedatabaseserver.
参数:
  参数              connection_id           ODBC 连接标识符，详见odbc_connect()。                  query_string                  The SQL statement.                        flags                  This parameter is currently not used.                     
返回值:
  返回值     Returns an ODBC result identifier if the SQL command was executed   successfully, or FALSE on error.
