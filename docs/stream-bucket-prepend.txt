语法:
   void stream_bucket_prepend    ( resource $brigade   , resource $bucket   )
说明:
Thisfunctioncanbecalledtoprependabuckettoabucketbrigade.Itistypicallycalledfromphp_user_filter::filter().
参数:
  参数         brigade               brigade is a resource pointing to a bucket brigade      which contains one or more bucket objects.                   bucket               A bucket object.               
返回值:
  返回值     没有返回值。   
示例:

<?php
class foo extends php_user_filter {
  protected $calls = 0;
  public function filter($in, $out, &$consumed, $closing) {
    while ($bucket = stream_bucket_make_writeable($in)) {
      $consumed += $bucket->datalen;
      if ($this->calls++ == 2) {
        // This bucket will appear again before any other bucket.
        stream_bucket_prepend($in, $bucket);
      }
    }
    return PSFS_FEED_ME;
  }
}
stream_filter_register('test', 'foo');
print  file_get_contents('php://filter/read=test/resource=foo');
?>
