语法:
   string get_called_class    ( void   )
说明:
获取静态方法调用的类名。
返回值:
  返回值     返回类的名称，如果不是在类中调用则返回 FALSE。   
示例:

<?php
class foo {
    static public function test() {
        var_dump(get_called_class());
    }
}
class bar extends foo {
}
foo::test();
bar::test();
?>
以上例程会输出：
string(3) "foo"
string(3) "bar"
