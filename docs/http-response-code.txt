语法:
   int http_response_code    ([ int $response_code  ] )
说明:
Ifyoupassnoparametersthenhttp_response_codewillgetthecurrentstatuscode.Ifyoupassaparameteritwillsettheresponsecode.
参数:
  参数              response_code                  The optional response_code will set the response code.                           &lt;?phphttp_response_code(404);?&gt;
返回值:
  返回值     The current response code. By default the return value is int(200).   
示例:

<?php
http_response_code(404);
?>

<?php
// Get the current default response code
var_dump(http_response_code()); // int(200)
// Set our response code
http_response_code(404);
// Get our new response code
var_dump(http_response_code()); // int(404)
?>
