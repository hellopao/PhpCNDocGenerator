语法:
   int filesize    ( string $filename   )
说明:
取得指定文件的大小。
参数:
  参数              filename                  文件的路径。                     
返回值:
  返回值     返回文件大小的字节数，如果出错返回 FALSE 并生成一条   E_WARNING 级的错误。    Note: 因为 PHP 的整数类型是有符号整型而且很多平台使用 32 位整型，对 2GB以上的文件，一些文件系统函数可能返回无法预期的结果 。 
示例:

<?php
// 输出类似：somefile.txt: 1024 bytes
$filename = 'somefile.txt';
echo $filename . ': ' . filesize($filename) . ' bytes';
?>
