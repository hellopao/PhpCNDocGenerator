语法:
   mixed constant    ( string $name   )
参数:
  参数              name                     常量名。                     
返回值:
  返回值    返回常量的值。如果常量未定义则返回 NULL。   
示例:

<?php
define("MAXSIZE", 100);
echo MAXSIZE;
echo constant("MAXSIZE"); // same thing as the previous line
interface bar {
    const test = 'foobar!';
}
class foo {
    const test = 'foobar!';
}
$const = 'test';
var_dump(constant('bar::'. $const)); // string(7) "foobar!"
var_dump(constant('foo::'. $const)); // string(7) "foobar!"
?>
