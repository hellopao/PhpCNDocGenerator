语法:
   array gethostbynamel    ( string $hostname   )
说明:
ReturnsalistofIPv4addressestowhichtheInternethostspecifiedbyhostnameresolves.
参数:
  参数              hostname                  The host name.                     
返回值:
  返回值     Returns an array of IPv4 addresses or FALSE if   hostname could not be resolved.   
示例:

<?php
$hosts = gethostbynamel('www.example.com');
print_r($hosts);
?>
以上例程会输出：
Array
(
    [0] => 192.0.34.166
)
