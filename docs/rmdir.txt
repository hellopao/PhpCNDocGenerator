语法:
   bool rmdir    ( string $dirname   [, resource $context  ] )
说明:
尝试删除dirname所指定的目录。该目录必须是空的，而且要有相应的权限。失败时会产生一个E_WARNING级别的错误。
参数:
  参数              dirname                  目录的路径。                        context           Note: 在 PHP 5.0.0中增加了对上下文（Context）的支持。有关上下文（Context）的说明参见Streams。               
返回值:
  返回值     成功时返回 TRUE， 或者在失败时返回 FALSE。   
示例:

<?php
if (!is_dir('examples')) {
    mkdir('examples');
}
rmdir('examples');
?>
