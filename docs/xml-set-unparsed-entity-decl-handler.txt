语法:
     bool xml_set_unparsed_entity_decl_handler      ( resource $parser   , callable $handler     )
说明:
为parser参数指定的XML解析器建立未解析实体定义声明处理器函数。
参数:
  参数              parser                  XML 解析器的引用，用于建立未解析实体定义声明处理器。                        handler                  参数handler 为表示函数名称的字符串，这个函数必须在为 parser 指定的解析器调用 xml_parse() 函数时已存在。                   由 handler 参数命名的函数名必须接受六个参数：               handler         ( resource $parser        , string $entity_name        , string $base        , string $system_id        , string $public_id        , string $notation_name        )
返回值:
  返回值     成功时返回 TRUE， 或者在失败时返回 FALSE。   
示例:

<!ENTITY <parameter>name</parameter> {<parameter>publicId</parameter> | <parameter>systemId</parameter>} 
        NDATA <parameter>notationName</parameter>
