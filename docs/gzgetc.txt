语法:
   string gzgetc    ( resource $zp   )
说明:
Returnsastringcontainingasingle(uncompressed)characterreadfromthegivengz-filepointer.
参数:
  参数              zp                  The gz-file pointer. It must be valid, and must point to a file       successfully opened by gzopen().                     
返回值:
  返回值     The uncompressed character or FALSE on EOF (unlike gzeof()).   
示例:

<?php
$gz = gzopen('somefile.gz', 'r');
while (!gzeof($gz)) {
  echo gzgetc($gz);
}
gzclose($gz);
?>
