语法:
   string inet_pton    ( string $address   )
说明:
ThisfunctionconvertsahumanreadableIPv4orIPv6address(ifPHPwasbuiltwithIPv6supportenabled)intoanaddressfamilyappropriate32bitor128bitbinarystructure.
参数:
  参数              address                  A human readable IPv4 or IPv6 address.                     
返回值:
  返回值     Returns the in_addr representation of the given   address, or FALSE if a syntactically invalid   address is given (for example, an IPv4 address   without dots or an IPv6 address without colons).   
示例:

 
<?php
$in_addr = inet_pton('127.0.0.1');
 
$in6_addr = inet_pton('::1');
?>
