语法:
   string ucfirst    ( string $str   )
说明:
将str的首字符（如果首字符是字母）转换为大写字母，并返回这个字符串。
参数:
  参数              str                  输入字符串。                     
返回值:
  返回值     返回结果字符串。   
示例:

<?php
$foo = 'hello world!';
$foo = ucfirst($foo);             // Hello world!
$bar = 'HELLO WORLD!';
$bar = ucfirst($bar);             // HELLO WORLD!
$bar = ucfirst(strtolower($bar)); // Hello world!
?>
