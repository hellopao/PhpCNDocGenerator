语法:
   bool defined    ( string $name   )
说明:
检查该名称的常量是否已定义。
参数:
  参数              name                  常量的名称。                     
返回值:
  返回值    如果该名称的常量已定义，返回 TRUE；未定义则返回 FALSE。   
示例:

<?php
/* Note the use of quotes, this is important.  This example is checking
 * if the string 'TEST' is the name of a constant named TEST */
if (defined('TEST')) {
    echo TEST;
}
?>
