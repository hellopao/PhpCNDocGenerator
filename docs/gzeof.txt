语法:
   int gzeof    ( resource $zp   )
说明:
TeststhegivenGZfilepointerforEOF.
参数:
  参数              zp                  The gz-file pointer. It must be valid, and must point to a file       successfully opened by gzopen().                     
返回值:
  返回值      Returns TRUE if the gz-file pointer is at EOF or an error occurs;   otherwise returns FALSE.   
示例:

<?php
$gz = gzopen('somefile.gz', 'r');
while (!gzeof($gz)) {
  echo gzgetc($gz);
}
gzclose($gz);
?>
