语法:
   int fileowner    ( string $filename   )
说明:
取得文件的所有者。
参数:
  参数              filename                  文件的路径。                     
返回值:
  返回值     返回文件所有的用户 ID，如果出错则返回 FALSE。用户   ID 以数字格式返回，用   posix_getpwuid() 来将其解析为用户名。   
示例:

<?php
$filename = 'index.php';
print_r(posix_getpwuid(fileowner($filename)));
?>
