语法:
   string mhash_get_hash_name    ( int $hash   )
说明:
Getsthenameofthespecifiedhash.
参数:
  参数              hash                  The hash ID. One of the MHASH_hashname constants.                     
返回值:
  返回值     Returns the name of the hash or FALSE, if the hash does not exist.   
示例:

<?php
echo mhash_get_hash_name(MHASH_MD5); // MD5
?>
