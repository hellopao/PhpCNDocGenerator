语法:
   string mcrypt_create_iv    ( int $size   [, int $source = MCRYPT_DEV_URANDOM  ] )
说明:
从随机源创建初始向量。
参数:
  参数              size                  初始向量大小。                        source                  初始向量数据来源。可选值有：       MCRYPT_RAND （系统随机数生成器）,       MCRYPT_DEV_RANDOM （从       /dev/random 文件读取数据） 和       MCRYPT_DEV_URANDOM （从       /dev/urandom 文件读取数据）。       在 Windows 平台，PHP 5.3.0 之前的版本中，仅支持 MCRYPT_RAND。                   请注意，在 PHP 5.6.0 之前的版本中，       此参数的默认值为 MCRYPT_DEV_RANDOM。                     
返回值:
  返回值     返回初始向量。如果发生错误，则返回 FALSE。   
示例:

<?php
    $size = mcrypt_get_iv_size(MCRYPT_CAST_256, MCRYPT_MODE_CFB);
    $iv = mcrypt_create_iv($size, MCRYPT_DEV_RANDOM);
?>
