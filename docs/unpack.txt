语法:
   array unpack    ( string $format   , string $data   )
说明:
Unpacksfromabinarystringintoanarrayaccordingtothegivenformat.
参数:
  参数              format                  See pack() for an explanation of the format codes.                        data                  The packed data.                     
返回值:
  返回值     Returns an associative array containing unpacked elements of binary   string.   
示例:

<?php
$binarydata = "\x04\x00\xa0\x00";
$array = unpack("cchars/nint", $binarydata);
?>

     The resulting array will contain the entries "chars" with value
     4 and "int" with 160.
    
<?php
$binarydata = "\x04\x00\xa0\x00";
$array = unpack("c2chars/nint", $binarydata);
?>

      The resulting array will contain the entries "chars1",
      "chars2" and "int".
     
<?php
$binarydata = "\x32\x42\x00\xa0";
$array = unpack("c2/n", $binarydata);
var_dump($array);
?>

      The resulting array will contain the entries "1" with value
      160 and "2" with 66. The
      first value from the c specifier is 
      overwritten by the first value from the n
      specifier.
