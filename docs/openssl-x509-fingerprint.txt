语法:
   bool openssl_x509_fingerprint    ( mixed $x509   [, string $hash_algorithm = &quot;sha1&quot;   [, bool $raw_output = FALSE  ]] )
说明:
openssl_x509_fingerprint()returnsthedigestofx509asastring.
参数:
  参数         x509   参见密钥／证书参数以获取有效值列表。          hash_algorithm                  The hash algorithm to use, e.g. &quot;md5&quot; or &quot;sha1&quot;                        raw_output                 When set to TRUE, outputs raw binary data. FALSE outputs lowercase hexits.                     
返回值:
  返回值      Returns a string containing the calculated certificate fingerprint as lowercase hexits unless raw_output is set to TRUE in which case the raw binary representation of the message digest is returned.        Returns FALSE on failure.
