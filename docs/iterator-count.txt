语法:
   int iterator_count    ( Traversable $iterator   )
说明:
计算迭代器中的元素个数。
参数:
  参数              iterator                  要计数的迭代器。                     
返回值:
  返回值     迭代器iterator中的元素个数。   
示例:

<?php
$iterator = new ArrayIterator(array('recipe'=>'pancakes', 'egg', 'milk', 'flour'));
var_dump(iterator_count($iterator));
?>
以上例程会输出：
int(4)
