语法:
   bool is_a    ( object $object   , string $class_name   [, bool $allow_string = FALSE  ] )
说明:
如果object是该类或该类是此对象的父类。
参数:
  参数              object                  The tested object                        class_name                  The class name                        allow_string                  If this parameter set to FALSE, string class name as object       is not allowed. This also prevents from calling autoloader if the class doesn&#039;t exist.                     
返回值:
  返回值     Returns TRUE if the object is of this class or has this class as one of   its parents, FALSE otherwise.   
示例:

<?php
// define a class
class WidgetFactory
{
  var $oink = 'moo';
}
// create a new object
$WF = new WidgetFactory();
if (is_a($WF, 'WidgetFactory')) {
  echo "yes, \$WF is still a WidgetFactory\n";
}
?>

<?php
if ($WF instanceof WidgetFactory) {
    echo 'Yes, $WF is a WidgetFactory';
}
?>
