语法:
   int ftp_nb_fget    ( resource $ftp_stream   , resource $handle   , string $remote_file   , int $mode   [, int $resumepos = 0  ] )
说明:
ftp_nb_fget()从远程FTP服务器获取一个文件
参数:
  参数              ftp_stream                  FTP 连接标示符。                        handle                  用来存储数据的一个已经打开的文件句柄。                        remote_file                  远程文件路径。                        mode                  传输模式，必须是 FTP_ASCII 或者       FTP_BINARY。                        resumepos           远程文件开始下载的位置（即从远程文件的哪个字节开始下载）。               
返回值:
  返回值      返回 FTP_FAILED 或 FTP_FINISHED   或 FTP_MOREDATA。   
示例:

<?php
// 打开要写入的文件
$file = 'index.php';
$fp = fopen($file, 'w');
$conn_id = ftp_connect($ftp_server);
$login_result = ftp_login($conn_id, $ftp_user_name, $ftp_user_pass);
// 初始化下载
$ret = ftp_nb_fget($conn_id, $fp, $file, FTP_BINARY);
while ($ret == FTP_MOREDATA) {
   // 其他要做的工作
   echo ".";
   // 继续下载...
   $ret = ftp_nb_continue($conn_id);
}
if ($ret != FTP_FINISHED) {
   echo "There was an error downloading the file...";
   exit(1);
}
// 关闭文件句柄
fclose($fp);
?>
