语法:
   int gregoriantojd    ( int $month   , int $day   , int $year   )
说明:
Gregorian历法的合理范围是4714B.C.至9999A.D.
参数:
  参数              month                  月份的范围是 1（January）到 12（December）。                        day                  日期的范围是 1到 31。                        year                  年份的范围是 -4714 到 9999。                       
返回值:
  返回值     给定gregorian历法日期的julian天数。   
示例:

<?php
$jd = GregorianToJD(10, 11, 1970);
echo "$jd\n";
$gregorian = JDToGregorian($jd);
echo "$gregorian\n";
?>
