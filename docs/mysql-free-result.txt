语法:
      bool mysql_free_result       ( resource $result      )
说明:
mysql_free_result()将释放所有与结果标识符result所关联的内存。
参数:
  参数         resultresource型的结果集。此结果集来自对 mysql_query()的调用。      
返回值:
  返回值     成功时返回 TRUE， 或者在失败时返回 FALSE。       If a non-resource is used for the result, an   error of level E_WARNING will be emitted.  It&#039;s worth noting that   mysql_query() only returns a resource   for SELECT, SHOW, EXPLAIN, and DESCRIBE queries.   
示例:

<?php
$result = mysql_query("SELECT id,email FROM people WHERE id = '42'");
if (!$result) {
    echo 'Could not run query: ' . mysql_error();
    exit;
}
/* Use the result, assuming we're done with it afterwards */
$row = mysql_fetch_assoc($result);
/* Now we free up the result and continue on with our script */
mysql_free_result($result);
echo $row['id'];
echo $row['email'];
?>
