语法:
   string system    ( string $command   [, int &$return_var  ] )
说明:
同C版本的system()函数一样，本函数执行command参数所指定的命令，并且输出执行结果。
参数:
  参数              command                  要执行的命令。                        return_var                   如果提供 return_var 参数，        则外部命令执行后的返回状态将会被设置到此变量中。                     
返回值:
  返回值      成功则返回命令输出的最后一行，    失败则返回 FALSE   
示例:

<?php
echo '<pre>';
// 输出 shell 命令 "ls" 的返回结果
// 并且将输出的最后一样内容返回到 $last_line。
// 将命令的返回值保存到 $retval。
$last_line = system('ls', $retval);
// 打印更多信息
echo '
</pre>
<hr />Last line of the output: ' . $last_line . '
<hr />Return value: ' . $retval;
?>
