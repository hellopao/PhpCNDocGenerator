语法:
   string bcdiv    ( string $left_operand = &quot;&quot;   , string $right_operand = &quot;&quot;   [, int $scale = int  ] )
说明:
Dividestheleft_operandbytheright_operand.
参数:
  参数              left_operand                  The left operand, as a string.                        right_operand                  The right operand, as a string.                   scale此可选参数用于设置结果中小数点后的小数位数。也可通过使用bcscale() 来设置全局默认的小数位数，用于所有函数。      
返回值:
  返回值     Returns the result of the division as a string, or NULL if    right_operand is 0.   
示例:

<?php
echo bcdiv('105', '6.55957', 3);  // 16.007
?>
