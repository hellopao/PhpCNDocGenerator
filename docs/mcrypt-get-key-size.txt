语法:
   int mcrypt_get_key_size    ( int $cipher   )
说明:
第一个原型是针对libmcrypt2.2.x的，第二个原型是针对libmcrypt2.4.x或2.5.x的。
参数:
  参数              cipher           MCRYPT_ciphername 常量中的一个，或者是字符串值的算法名称。                  mode           MCRYPT_MODE_modename 常量中的一个，或以下字符串中的一个：&quot;ecb&quot;，&quot;cbc&quot;，&quot;cfb&quot;，&quot;ofb&quot;，&quot;nofb&quot; 和 &quot;stream&quot;。               
返回值:
  返回值     返回算法所支持的最大密钥大小，以字节为单位。    或者在失败时返回 FALSE。   
示例:

<?php
    echo mcrypt_get_key_size('tripledes', 'ecb');
?>

     在 libmcrypt 2.4.x 或 2.5.x 版本中，
     如果使用本函数。
    以上例程会输出：
24
