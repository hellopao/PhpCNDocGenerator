语法:
   int getservbyname    ( string $service   , string $protocol   )
说明:
getservbyname()returnstheInternetportwhichcorrespondstoserviceforthespecifiedprotocolasper/etc/services.
参数:
  参数              service                  The Internet service name, as a string.                        protocol                  protocol is either &quot;tcp&quot;       or &quot;udp&quot; (in lowercase).                     
返回值:
  返回值     Returns the port number, or FALSE if service or   protocol is not found.   
示例:

<?php
$services = array('http', 'ftp', 'ssh', 'telnet', 'imap',
'smtp', 'nicname', 'gopher', 'finger', 'pop3', 'www');
foreach ($services as $service) {
    $port = getservbyname($service, 'tcp');
    echo $service . ": " . $port . "<br />\n";
}
?>
