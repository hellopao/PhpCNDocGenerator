语法:
   string stream_resolve_include_path    ( string $filename      )
说明:
Resolvefilenameagainsttheincludepathaccordingtothesamerulesasfopen()/include.
参数:
  参数              filename                  The filename to resolve.                         
返回值:
  返回值     Returns a string containing the resolved absolute filename,  或者在失败时返回 FALSE.   
示例:

     Basic usage example.
    
<?php
var_dump(stream_resolve_include_path("test.php"));
?>
以上例程的输出类似于：
string(22) "/var/www/html/test.php"
