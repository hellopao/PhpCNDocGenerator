语法:
   int strncasecmp    ( string $str1   , string $str2   , int $len   )
说明:
该函数与strcasecmp()类似，不同之处在于你可以指定两个字符串比较时使用的长度（即最大比较长度）。
参数:
  参数              str1                  第一个字符串。                        str2                  第二个字符串。                        len                  最大比较长度。                     
返回值:
  返回值     如果 str1 小于   str2 返回 &lt; 0；   如果 str1   大于 str2   返回 &gt; 0；如果两者相等，返回 0。
