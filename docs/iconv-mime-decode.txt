语法:
   string iconv_mime_decode    ( string $encoded_header   [, int $mode = 0   [, string $charset = ini_get(&quot;iconv.internal_encoding&quot;)  ]] )
说明:
DecodesaMIMEheaderfield.
参数:
  参数              encoded_header                  The encoded header, as a string.                        mode                  mode determines the behaviour in the event       iconv_mime_decode() encounters a malformed       MIME header field. You can specify any combination       of the following bitmasks.               Bitmasks acceptable to iconv_mime_decode()                                      Value           Constant           Description                                                 1           ICONV_MIME_DECODE_STRICT                       If set, the given header is decoded in full conformance with the            standards defined in &raquo;&nbsp;RFC2047.            This option is disabled by default because there are a lot of            broken mail user agents that don&#039;t follow the specification and don&#039;t            produce correct MIME headers.                                          2           ICONV_MIME_DECODE_CONTINUE_ON_ERROR                       If set, iconv_mime_decode_headers()            attempts to ignore any grammatical errors and continue to process            a given header.                                                                     charset                  The optional charset parameter specifies the       character set to represent the result by. If omitted,       iconv.internal_encoding       will be used.                     
返回值:
  返回值     Returns a decoded MIME field on success,   or FALSE if an error occurs during the decoding.   
示例:

<?php
// This yields "Subject: Prüfung Prüfung"
echo iconv_mime_decode("Subject: =?UTF-8?B?UHLDvGZ1bmcgUHLDvGZ1bmc=?=",
                       0, "ISO-8859-1");
?>
