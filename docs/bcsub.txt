语法:
   string bcsub    ( string $left_operand = &quot;&quot;   , string $right_operand = &quot;&quot;   [, int $scale = int  ] )
说明:
Subtractstheright_operandfromtheleft_operand.
参数:
  参数              left_operand                  The left operand, as a string.                        right_operand                  The right operand, as a string.                   scale此可选参数用于设置结果中小数点后的小数位数。也可通过使用bcscale() 来设置全局默认的小数位数，用于所有函数。      
返回值:
  返回值     The result of the subtraction, as a string.   
示例:

<?php
$a = '1.234';
$b = '5';
echo bcsub($a, $b);     // -3
echo bcsub($a, $b, 4);  // -3.7660
?>
