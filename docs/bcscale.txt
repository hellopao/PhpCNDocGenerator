语法:
   bool bcscale    ( int $scale   )
说明:
Setsthedefaultscaleparameterforallsubsequentcallstobcmathfunctionsthatdonotexplicitlyspecifyascaleparameter.
参数:
  参数              scale                  The scale factor.                     
返回值:
  返回值     成功时返回 TRUE， 或者在失败时返回 FALSE。   
示例:

<?php
// default scale : 3
bcscale(3);
echo bcdiv('105', '6.55957'); // 16.007
// this is the same without bcscale()
echo bcdiv('105', '6.55957', 3); // 16.007
?>
