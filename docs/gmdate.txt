语法:
   string gmdate    ( string $format   [, int $timestamp  ] )
说明:
同date()函数完全一样，只除了返回的时间是格林威治标准时（GMT）。例如当在中国（GMT+0800）运行以下程序时，第一行显示“Jan01200000:00:00”而第二行显示“Dec31199916:00:00”。Example#1gmdate()例子
示例:

<?php
echo date("M d Y H:i:s", mktime (0,0,0,1,1,2000));
echo gmdate("M d Y H:i:s", mktime (0,0,0,1,1,2000));
?>
