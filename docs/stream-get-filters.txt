语法:
   array stream_get_filters    ( void   )
返回值:
  返回值     返回一个包含所有有效的数据流过滤器名字的索引数组。   
示例:

<?php
$streamlist = stream_get_filters();
print_r($streamlist);
?>
以上例程的输出类似于：
Array (
  [0] => string.rot13
  [1] => string.toupper
  [2] => string.tolower
  [3] => string.base64
  [4] => string.quoted-printable
)
