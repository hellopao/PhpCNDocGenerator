语法:
    number pow     ( number $base    , number $exp    )
说明:
返回base的exp次方的幂。如果可能，本函数会返回integer。
参数:
  参数              base                  The base to use                        exp                  指数                     
返回值:
  返回值       base 的     exp 次方的幂。   If both arguments are non-negative integers and the result can be represented   as an integer, the result will be returned with integer type,   otherwise it will be returned as a float.   
示例:

<?php
var_dump(pow(2, 8)); // int(256)
echo pow(-1, 20); // 1
echo pow(0, 0); // 1
echo pow(-1, 5.5); // PHP >4.0.6  NAN
echo pow(-1, 5.5); // PHP <=4.0.6 1.#IND
?>
