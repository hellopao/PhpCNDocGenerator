语法:
      number array_sum       ( array $array      )
说明:
array_sum()将数组中的所有值的和以整数或浮点数的结果返回。
参数:
  参数              array                  输入的数组。                     
返回值:
  返回值     所有值的和以整数或浮点数的结果返回。   
示例:

<?php
$a = array(2, 4, 6, 8);
echo "sum(a) = " . array_sum($a) . "\n";
$b = array("a" => 1.2, "b" => 2.3, "c" => 3.4);
echo "sum(b) = " . array_sum($b) . "\n";
?>
以上例程会输出：
sum(a) = 20
sum(b) = 6.9
