语法:
   string ucwords    ( string $str   )
说明:
将str中每个单词的首字符（如果首字符是字母）转换为大写字母，并返回这个字符串。
参数:
  参数              str                  输入字符串。                     
返回值:
  返回值     返回转换后的字符串。   
示例:

<?php
$foo = 'hello world!';
$foo = ucwords($foo);             // Hello World!
$bar = 'HELLO WORLD!';
$bar = ucwords($bar);             // HELLO WORLD!
$bar = ucwords(strtolower($bar)); // Hello World!
?>
