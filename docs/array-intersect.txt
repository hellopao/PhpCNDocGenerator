语法:
   array array_intersect    ( array $array1   , array $array2   [, array $ ...  ] )
说明:
array_intersect()返回一个数组，该数组包含了所有在array1中也同时出现在所有其它参数数组中的值。注意键名保留不变。
参数:
  参数              array1                  The array with master values to check.                        array2                  An array to compare values against.                        array                  A variable list of arrays to compare.                     
返回值:
  返回值      返回一个数组，该数组包含了所有在     array1 中也同时出现在所有其它参数数组中的值。   
示例:

<?php
$array1 = array("a" => "green", "red", "blue");
$array2 = array("b" => "green", "yellow", "red");
$result = array_intersect($array1, $array2);
print_r($result);
?>
以上例程会输出：
Array
(
    [a] => green
    [0] => red
)
