语法:
   array mcrypt_list_algorithms    ([ string $lib_dir = ini_get(&quot;mcrypt.algorithms_dir&quot;)  ] )
说明:
获取lib_dir中包含的受支持的算法。
参数:
  参数              lib_dir                  指定算法所在的位置。       如果未指定，将使用 php.ini       中的 mcrypt.algorithms_dir 指示所指定的位置。                     
返回值:
  返回值     以数组方式返回所有受支持的算法。   
示例:

<?php
    $algorithms = mcrypt_list_algorithms("/usr/local/lib/libmcrypt");
    foreach ($algorithms as $cipher) {
        echo "$cipher<br />\n";
    }
?>

     本例程返回 "/usr/local/lib/libmcrypt"
     目录下所有受支持的算法。
