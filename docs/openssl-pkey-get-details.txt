语法:
   array openssl_pkey_get_details    ( resource $key   )
说明:
Thisfunctionreturnsthekeydetails(bits,key,type).
参数:
  参数              key                  Resource holding the key.                     
返回值:
  返回值     Returns an array with the key details in success or FALSE in failure.   Returned array has indexes bits (number of bits),   key (string representation of the public key) and   type (type of the key which is one of   OPENSSL_KEYTYPE_RSA,   OPENSSL_KEYTYPE_DSA,   OPENSSL_KEYTYPE_DH,   OPENSSL_KEYTYPE_EC or -1 meaning unknown).       Depending on the key type used, additional details may be returned. Note that    some elements may not always be available.                OPENSSL_KEYTYPE_RSA, an additional array key named &quot;rsa&quot;,      containing the key data is returned.
