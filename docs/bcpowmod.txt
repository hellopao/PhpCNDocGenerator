语法:
   string bcpowmod    ( string $left_operand = &quot;&quot;   , string $right_operand = &quot;&quot;   , string $modulus = &quot;&quot;   [, int $scale = int  ] )
说明:
Usethefast-exponentiationmethodtoraiseleft_operandtothepowerright_operandwithrespecttothemodulusmodulus.
参数:
  参数              left_operand                  The left operand, as a string.                        right_operand                  The right operand, as a string.                        modulus                  The modulus, as a string.                   scale此可选参数用于设置结果中小数点后的小数位数。也可通过使用bcscale() 来设置全局默认的小数位数，用于所有函数。      
返回值:
  返回值     Returns the result as a string, or NULL if modulus   is 0.   
示例:

<?php
$a = bcpowmod($x, $y, $mod);
$b = bcmod(bcpow($x, $y), $mod);
// $a and $b are equal to each other.
?>
