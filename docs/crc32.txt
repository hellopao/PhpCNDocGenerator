语法:
   int crc32    ( string $str   )
说明:
生成str的32位循环冗余校验码多项式。这通常用于检查传输的数据是否完整。
参数:
  参数                  str                     要校验的数据。                          
返回值:
  返回值       返回 str crc32 校验的整数。    
示例:

     示例中的第二行演示了如何使用 printf()
     函数转换校验码：
    
<?php
$checksum = crc32("The quick brown fox jumped over the lazy dog.");
printf("%u\n", $checksum);
?>
