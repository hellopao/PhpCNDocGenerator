语法:
   int vprintf    ( string $format   , array $args   )
说明:
根据format（sprintf()函数文档中有相关描述）参数指定的格式，在一个格式化字符串中显示多个值。
参数:
  参数              format                  关于 format 的描述，参见 sprintf()。                        args                                
返回值:
  返回值     返回输出字符串的长度。   
示例:

<?php
vprintf("%04d-%02d-%02d", explode('-', '1988-8-1')); // 1988-08-01
?>
