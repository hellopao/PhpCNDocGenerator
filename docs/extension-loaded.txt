语法:
   bool extension_loaded    ( string $name   )
说明:
检查一个扩展是否已经加载。
参数:
  参数              name                  扩展名称。                   你可以用 phpinfo() 来查看一系列扩展名称，而在 CGI 或 CLI 的 PHP 版本里你可以使用 -m 参数来列出所有有效的扩展：       $ php -m[PHP Modules]xmltokenizerstandardsocketssessionposixpcreoverloadmysqlmbstringctype[Zend Modules]
返回值:
  返回值     如果 name 指定的扩展已加载，返回TRUE，否则返回 FALSE。   
示例:

$ php -m
[PHP Modules]
xml
tokenizer
standard
sockets
session
posix
pcre
overload
mysql
mbstring
ctype
[Zend Modules]

<?php
if (!extension_loaded('gd')) {
    if (!dl('gd.so')) {
        exit;
    }
}
?>
