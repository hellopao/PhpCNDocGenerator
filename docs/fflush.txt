语法:
   bool fflush    ( resource $handle   )
说明:
本函数强制将所有缓冲的输出写入handle文件句柄所指向的资源。成功时返回TRUE，或者在失败时返回FALSE。
参数:
  参数              handle           文件指针必须是有效的，必须指向由fopen() 或 fsockopen() 成功打开的文件(并还未由 fclose() 关闭)。               
返回值:
  返回值     成功时返回 TRUE， 或者在失败时返回 FALSE。   
示例:

<?php
$filename = 'bar.txt';
$file = fopen($filename, 'r+');
rewind($file);
fwrite($file, 'Foo');
fflush($file);
ftruncate($file, ftell($file));
fclose($file);
?>
