语法:
   bool shuffle    ( array &$array   )
说明:
本函数打乱（随机排列单元的顺序）一个数组。
参数:
  参数              array                  待操作的数组。                     
返回值:
  返回值     成功时返回 TRUE， 或者在失败时返回 FALSE。   
示例:

<?php
$numbers = range(1, 20);
shuffle($numbers);
foreach ($numbers as $number) {
    echo "$number ";
}
?>
