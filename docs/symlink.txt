语法:
   bool symlink    ( string $target   , string $link   )
说明:
symlink()对于已有的target建立一个名为link的符号连接。
参数:
  参数              target                  连接的目标。                        link                  连接的名称。                     
返回值:
  返回值     成功时返回 TRUE， 或者在失败时返回 FALSE。   
示例:

<?php
$target = 'uploads.php';
$link = 'uploads';
symlink($target, $link);
echo readlink($link);
?>
