语法:
   string bcmod    ( string $left_operand   , string $modulus   )
说明:
Getthemodulusoftheleft_operandusingmodulus.
参数:
  参数              left_operand                  The left operand, as a string.                        modulus                  The modulus, as a string.                     
返回值:
  返回值     Returns the modulus as a string, or NULL if    modulus is 0.   
示例:

<?php
echo bcmod('4', '2'); // 0
echo bcmod('2', '4'); // 2
?>
