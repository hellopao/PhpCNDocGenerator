语法:
   number array_product    ( array $array   )
说明:
array_product()以整数或浮点数返回一个数组中所有值的乘积。
参数:
  参数              array                  这个数组。                     
返回值:
  返回值     以整数或浮点数返回一个数组中所有值的乘积。   
示例:

<?php
$a = array(2, 4, 6, 8);
echo "product(a) = " . array_product($a) . "\n";
echo "product(array()) = " . array_product(array()) . "\n";
?>
以上例程会输出：
product(a) = 384
product(array()) = 1
