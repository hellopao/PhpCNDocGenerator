语法:
   void restore_include_path    ( void   )
返回值:
  返回值     没有返回值。   
示例:

<?php
echo get_include_path();  // .:/usr/local/lib/php
set_include_path('/inc');
echo get_include_path();  // /inc
// 自 PHP 4.3.0 起可用
restore_include_path();
// 所有版本均可用
ini_restore('include_path');
echo get_include_path();  // .:/usr/local/lib/php
?>
