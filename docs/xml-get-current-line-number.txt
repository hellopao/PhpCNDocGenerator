语法:
     int xml_get_current_line_number      ( resource $parser     )
说明:
获取指定XML解析器当前的行号。
参数:
    参数                      parser                        一个指向要获取当前行号的 XML 解析器的指针。                                 
返回值:
    返回值         如果 parser 参数没有指向一个合法的解析器，该函数将返回 FALSE，否则将返回指定解析器在其缓存中的当前行号。
