语法:
   bool feof    ( resource $handle   )
说明:
测试文件指针是否到了文件结束的位。
参数:
  参数              handle           文件指针必须是有效的，必须指向由fopen() 或 fsockopen() 成功打开的文件(并还未由 fclose() 关闭)。               
返回值:
  返回值     如果文件指针到了 EOF 或者出错时则返回 TRUE，否则返回一个错误（包括 socket   超时），其它情况则返回 FALSE。   
示例:

<?php
function safe_feof($fp, &$start = NULL) {
 $start = microtime(true);
 return feof($fp);
}
/* $fp 的赋值是由之前 fsockopen() 打开  */
$start = NULL;
$timeout = ini_get('default_socket_timeout');
while(!safe_feof($fp, $start) && (microtime(true) - $start) < $timeout)
{
 /* Handle */
}
?>

<?php
// 如果文件不可读取或者不存在，fopen 函数返回 FALSE
$file = @fopen("no_such_file", "r");
// 来自 fopen 的 FALSE 会发出一条警告信息并在这里陷入无限循环
while (!feof($file)) {
}
fclose($file);
?>
