语法:
   resource stream_context_set_default    ( array $options   )
参数:
  参数              options                  The options to set for the default context.            Note:                options must be an associative        array of associative arrays in the format        $arr[&#039;wrapper&#039;][&#039;option&#039;] = $value.                            
返回值:
  返回值     Returns the default stream context.    
示例:

<?php
$default_opts = array(
  'http'=>array(
    'method'=>"GET",
    'header'=>"Accept-language: en\r\n" .
              "Cookie: foo=bar",
    'proxy'=>"tcp://10.54.1.39:8000"
  )
);
$default = stream_context_set_default($default_opts);
/* Sends a regular GET request to proxy server at 10.54.1.39
 * For www.example.com using context options specified in $default_opts
 */
readfile('http://www.example.com');
?>
