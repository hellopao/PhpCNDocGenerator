语法:
   int odbc_fetch_into    ( resource $result_id   , array &$result_array   [, int $rownumber  ] )
说明:
Fetchoneresultrowintoarray.
参数:
  参数              result_id                  The result resource.                        result_array                  The result array       that can be of any type since it will be converted to type       array. The array will contain the column values starting at array       index 0.                        rownumber                  The row number.                     
返回值:
  返回值     Returns the number of columns in the result;   FALSE on error.   
示例:

<?php
$rc = odbc_fetch_into($res_id, $my_array);
?>

     or
    
<?php
$rc = odbc_fetch_into($res_id, $my_array, 2);
?>
