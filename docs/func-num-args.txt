语法:
   int func_num_args    ( void   )
说明:
Getsthenumberofargumentspassedtothefunction.
返回值:
  返回值     Returns the number of arguments passed into the current user-defined   function.   
示例:

<?php
function foo()
{
    $numargs = func_num_args();
    echo "Number of arguments: $numargs\n";
}
foo(1, 2, 3);   
?>
以上例程会输出：
Number of arguments: 3

test.php
<?php
function foo() {
    include './fna.php';
}
foo('First arg', 'Second arg');
?>
