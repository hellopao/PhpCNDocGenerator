语法:
   string convert_uudecode    ( string $data   )
说明:
convert_uudecode()解码一个uuencode编码的字符串。
参数:
  参数              data                  uuencode 编码后的数据                     
返回值:
  返回值     返回解码后的字符串数据， 或者在失败时返回 FALSE.。   
示例:

<?php
/* 你猜会输出啥？:) */
echo convert_uudecode("+22!L;W9E(%!(4\"$`\n`");
?>
