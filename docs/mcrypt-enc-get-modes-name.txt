语法:
   string mcrypt_enc_get_modes_name    ( resource $td   )
说明:
返回模式名称。
参数:
  参数              td                  加密描述符。                     
返回值:
  返回值     以字符串格式返回模式名称。   
示例:

<?php
$td = mcrypt_module_open (MCRYPT_CAST_256, '', MCRYPT_MODE_CFB, '');
echo mcrypt_enc_get_modes_name($td). "\n";
$td = mcrypt_module_open ('cast-256', '', 'ecb', '');
echo mcrypt_enc_get_modes_name($td). "\n";
?>
以上例程会输出：
CFB
ECB
