语法:
   mixed gettimeofday    ([ bool $return_float = false  ] )
说明:
本函数是gettimeofday(2)的接口。返回一个关联数组，包含有系统调用返回的数据。
参数:
  参数                  return_float                     当其设为        TRUE 时，会返回一个浮点数而不是一个数组。                          
返回值:
  返回值         默认返回一个 array。如果  return_float     设置了则会返回一个 float。             数组中的键为：                          &quot;sec&quot; - 自 Unix 纪元起的秒数                                  &quot;usec&quot; - 微秒数                                  &quot;minuteswest&quot; - 格林威治向西的分钟数                                          &quot;dsttime&quot; - 夏令时修正的类型                       
示例:

<?php
print_r(gettimeofday());
echo gettimeofday(true);
?>
以上例程的输出类似于：
Array
(
    [sec] => 1073504408
    [usec] => 238215
    [minuteswest] => 0
    [dsttime] => 1
)
1073504408.23910
