语法:
      array array_unique       ( array $array    [, int $sort_flags = SORT_STRING     ] )
说明:
array_unique()接受array作为输入并返回没有重复值的新数组。
参数:
  参数              array                  输入的数组。                        sort_flags                  The optional second parameter sort_flags       may be used to modify the sorting behavior using these values:                   Sorting type flags:                        SORT_REGULAR - compare items normally         (don&#039;t change types)                         SORT_NUMERIC - compare items numerically                         SORT_STRING - compare items as strings                         SORT_LOCALE_STRING - compare items as          strings, based on the current locale.                                                       
返回值:
  返回值     Returns the filtered array.   
示例:

<?php
$input = array("a" => "green", "red", "b" => "green", "blue", "red");
$result = array_unique($input);
print_r($result);
?>
以上例程会输出：
Array
(
    [a] => green
    [0] => red
    [1] => blue
)

<?php
$input = array(4, "4", "3", 4, 3, "3");
$result = array_unique($input);
var_dump($result);
?>
以上例程会输出：
array(2) {
  [0] => int(4)
  [2] => string(1) "3"
}
