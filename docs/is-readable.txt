语法:
   bool is_readable    ( string $filename   )
说明:
判断给定文件名是否存在并且可读。
参数:
  参数              filename                  文件的路径。                     
返回值:
  返回值     如果由 filename   指定的文件或目录存在并且可读则返回 TRUE，否则返回 FALSE。   
示例:

<?php
$filename = 'test.txt';
if (is_readable($filename)) {
    echo 'The file is readable';
} else {
    echo 'The file is not readable';
}
?>
