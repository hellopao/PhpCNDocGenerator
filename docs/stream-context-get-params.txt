语法:
   array stream_context_get_params    ( resource $stream_or_context   )
说明:
Retrievesparameterandoptionsinformationfromthestreamorcontext.
参数:
  参数              stream_or_context                  A stream resource or a       context resource                     
返回值:
  返回值     Returns an associate array containing all context options and parameters.   
示例:

     Basic usage example
    
<?php
$ctx = stream_context_create();
$params = array("notification" => "stream_notification_callback");
stream_context_set_params($ctx, $params);
var_dump(stream_context_get_params($ctx));
?>
以上例程的输出类似于：
array(2) {
  ["notification"]=>
  string(28) "stream_notification_callback"
  ["options"]=>
  array(0) {
  }
}
