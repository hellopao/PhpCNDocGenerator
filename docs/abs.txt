语法:
    number abs     ( mixed $number    )
说明:
返回参数number的绝对值。
参数:
  参数              number                  要处理的数字值                     
返回值:
  返回值     number 的绝对值。 如果参数 number     是 float，则返回的类型也是 float，否则返回     integer（因为 float 通常比 integer 有更大的取值范围）。   
示例:

<?php
$abs = abs(-4.2); // $abs = 4.2; (double/float)
$abs2 = abs(5);   // $abs2 = 5; (integer)
$abs3 = abs(-5);  // $abs3 = 5; (integer)
?>
