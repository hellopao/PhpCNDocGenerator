语法:
   string hebrevc    ( string $hebrew_text   [, int $max_chars_per_line = 0  ] )
说明:
本函数与hebrev()一样，唯一的区别是本函数会额外将换行符(\n)转换为&quot;&lt;br&gt;\n&quot;。
参数:
  参数              hebrew_text                  逻辑顺序希伯来文字符串。                        max_chars_per_line                  可选参数，表示每行可返回的最多字符数。                     
返回值:
  返回值     返回视觉顺序字符串。
