语法:
   string gethostbyaddr    ( string $ip_address   )
说明:
返回指定的IP地址（ip_address）对应的主机名。
参数:
  参数              ip_address                  主机的IP地址。                     
返回值:
  返回值     成功则返回主机名；失败则原样输出（输出IP地址）；如果输入的格式不正常，则返回FALSE。   
示例:

<?php
$hostname = gethostbyaddr($_SERVER['REMOTE_ADDR']);
echo $hostname;
?>
