语法:
   string hash_final    ( resource $context   [, bool $raw_output = false  ] )
参数:
  参数              context                  hash_init() 函数返回的哈希运算上下文资源。                        raw_output                  设置为 TRUE，输出格式为原始的二进制数据。       设置为 FALSE，输出小写的 16 进制字符串。                     
返回值:
  返回值     如果 raw_output 设置为 TRUE， 则返回原始二进制数据表示的信息摘要，   否则返回 16 进制小写字符串格式表示的信息摘要。   
示例:

<?php
$ctx = hash_init('sha1');
hash_update($ctx, 'The quick brown fox jumped over the lazy dog.');
echo hash_final($ctx);
?>
以上例程会输出：
c0854fb9fb03c41cce3802cb0d220529e6eef94e
