语法:
   array array_fill_keys    ( array $keys   , mixed $value   )
说明:
使用value参数的值作为值，使用keys数组的值作为键来填充一个数组。
参数:
  参数              keys                  使用该数组的值作为键。非法值将被转换为字符串。                        value                  填充使用的值。                     
返回值:
  返回值     返回填充后的数组。   
示例:

<?php
$keys = array('foo', 5, 10, 'bar');
$a = array_fill_keys($keys, 'banana');
print_r($a);
?>
以上例程会输出：
Array
(
    [foo] => banana
    [5] => banana
    [10] => banana
    [bar] => banana
)
