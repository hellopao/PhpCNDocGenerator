语法:
     int xml_get_current_column_number      ( resource $parser     )
说明:
获得指定XML解析器当前的列号。
参数:
    参数                      parser                        一个指向要获取列号的 XML 解析器的指针。                                 
返回值:
    返回值         如果 parser 参数没有指向一个合法的解析器，该函数将返回 FALSE，否则将返回指定解析器所在行（由函数 xml_get_current_line_number() 给定）的当前列号。
