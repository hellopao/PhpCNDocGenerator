语法:
   array getimagesizefromstring    ( string $imagedata   [, array &$imageinfo  ] )
说明:
同getimagesize()函数。区别是getimagesizefromstring()第一个参数是图像数据的字符串表达，而不是文件名。
参数:
  参数         imagedata               图像数据的字符串表示。                   imageinfo               参见 getimagesize() 函数。               
返回值:
  返回值     参见 getimagesize() 函数。   
示例:

<?php
$img = '/path/to/test.png';
// 以文件方式打开
$size_info1 = getimagesize($img);
// 以字符串格式打开
$data       = file_get_contents($img);
$size_info2 = getimagesizefromstring($data);
?>
