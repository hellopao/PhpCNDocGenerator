语法:
   bool ctype_cntrl    ( string $text   )
说明:
检查提供的string和text里面的字符是不是都是控制字符。控制字符就是例如：换行、缩进、空格。
参数:
  参数              text                  需要被测试的字符串。                     
返回值:
  返回值     如果在当前的语言环境下 text 里面的每个字符都是控制字符，就返回 TRUE ；反之就返回 FALSE 。   
示例:

<?php
$strings = array('string1' => "\n\r\t", 'string2' => 'arf12');
foreach ($strings as $name => $testcase) {
    if (ctype_cntrl($testcase)) {
        echo "The string '$name' consists of all control characters.\n";
    } else {
        echo "The string '$name' does not consist of all control characters.\n";
    }
}
?>
以上例程会输出：
The string &#039;string1&#039; consists of all control characters.
The string &#039;string2&#039; does not consist of all control characters.
