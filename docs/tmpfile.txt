语法:
   resource tmpfile    ( void   )
说明:
以读写（w+）模式建立一个具有唯一文件名的临时文件，返回一个文件句柄。
返回值:
  返回值     返回一个与 fopen() 所打开返回类似的新文件句柄， 或者在失败时返回 FALSE.   
示例:

<?php
$temp = tmpfile();
fwrite($temp, "writing to tempfile");
fseek($temp, 0);
echo fread($temp, 1024);
fclose($temp); // this removes the file
?>
以上例程会输出：
writing to tempfile
