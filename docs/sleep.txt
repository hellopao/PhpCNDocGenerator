语法:
   int sleep    ( int $seconds   )
参数:
  参数              seconds                  暂停的秒数。                     
返回值:
  返回值     成功时返回 0，错误时返回 FALSE。       如果函数的调用被一个信号中止，sleep() 会返回一个非零的值。在Windows上，该值总是 192（即Windows API常量WAIT_IO_COMPLETION的值）。其他平台上，该返回值是剩余未sleep的秒数。   
示例:

<?php
// current time
echo date('h:i:s') . "\n";
// sleep for 10 seconds
sleep(10);
// wake up !
echo date('h:i:s') . "\n";
?>

     该例子会在休眠10秒后输出。
    
05:31:23
05:31:33
