语法:
   bool openssl_x509_export    ( mixed $x509   , string &$output   [, bool $notext = TRUE  ] )
说明:
openssl_x509_export()storesx509intoastringnamedbyoutputinaPEMencodedformat.
参数:
  参数         x509   参见密钥／证书参数以获取有效值列表。          output                  On success, this will hold the PEM.                        notext           可选参数 notext 影响输出的冗余度。如果设为FALSE，输出内容将包含附加的人类可读信息。notext 的缺省值为 TRUE。               
返回值:
  返回值     成功时返回 TRUE， 或者在失败时返回 FALSE。
