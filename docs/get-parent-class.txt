语法:
   string get_parent_class    ([ mixed $obj  ] )
说明:
如果obj是对象，则返回对象实例obj所属类的父类名。
参数:
  参数              object                  The tested object or class name                     
返回值:
  返回值     Returns the name of the parent class of the class of which   object is an instance or the name.    Note:        If the object does not have a parent or the class given does not exist FALSE will be returned.          If called without parameter outside object, this function returns FALSE.   
示例:

<?php
class dad {
    function dad()
    {
    // implements some logic
    }
}
class child extends dad {
    function child()
    {
        echo "I'm " , get_parent_class($this) , "'s son\n";
    }
}
class child2 extends dad {
    function child2()
    {
        echo "I'm " , get_parent_class('child2') , "'s son too\n";
    }
}
$foo = new child();
$bar = new child2();
?>
以上例程会输出：
I&#039;m dad&#039;s son
I&#039;m dad&#039;s son too
