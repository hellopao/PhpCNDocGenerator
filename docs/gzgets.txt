语法:
   string gzgets    ( resource $zp   [, int $length  ] )
说明:
Getsa(uncompressed)stringofuptolength-1bytesreadfromthegivenfilepointer.Readingendswhenlength-1byteshavebeenread,onanewline,oronEOF(whichevercomesfirst).
参数:
  参数              zp                  The gz-file pointer. It must be valid, and must point to a file       successfully opened by gzopen().                        length                  The length of data to get.                     
返回值:
  返回值     The uncompressed string, or FALSE on error.   
示例:

<?php
$handle = gzopen('somefile.gz', 'r');
while (!gzeof($handle)) {
   $buffer = gzgets($handle, 4096);
   echo $buffer;
}
gzclose($handle);
?>
