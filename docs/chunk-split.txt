语法:
   string chunk_split    ( string $body   [, int $chunklen = 76   [, string $end = &quot;\r\n&quot;  ]] )
说明:
使用此函数将字符串分割成小块非常有用。例如将base64_encode()的输出转换成符合RFC2045语义的字符串。它会在每chunklen个字符后边插入end。
参数:
  参数              body                  要分割的字符。                        chunklen                  分割的尺寸。                        end                  行尾序列符号。                     
返回值:
  返回值     返回分割后的字符。   
示例:

<?php
// 使用 RFC 2045 语义格式化 $data
$new_string = chunk_split(base64_encode($data));
?>
