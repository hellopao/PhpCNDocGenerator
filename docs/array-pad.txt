语法:
   array array_pad    ( array $input   , int $pad_size   , mixed $pad_value   )
说明:
array_pad()返回input的一个拷贝，并用pad_value将其填补到pad_size指定的长度。如果pad_size为正，则填补到数组的右侧，如果为负则从左侧开始填补。如果pad_size的绝对值小于或等于input数组的长度则没有任何填补。有可能一次最多填补1048576个单元。
参数:
  参数              input                  需要被填充的原始数组。                        pad_size                   新数组的长度。                        pad_value                  将被填充的值，只有在 input 的现有长度小于       pad_size 的长度时才有效。                     
返回值:
  返回值     返回 input 用  pad_value 填充到 pad_size 指定的长度之后的一个副本。   如果 pad_size 为正，则填补到数组的右侧，如果为负则从左侧开始填补。   如果 pad_size 的绝对值小于或等于 input 数组的长度则没有任何填补。   
示例:

<?php
$input = array(12, 10, 9);
$result = array_pad($input, 5, 0);
// result is array(12, 10, 9, 0, 0)
$result = array_pad($input, -7, -1);
// result is array(-1, -1, -1, -1, 12, 10, 9)
$result = array_pad($input, 2, "noop");
// not padded
?>
