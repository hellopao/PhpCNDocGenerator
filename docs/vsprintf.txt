语法:
   string vsprintf    ( string $format   , array $args   )
说明:
作用与sprintf()函数类似，但是接收一个数组参数，而不是一系列可变数量的参数。
参数:
  参数              format                  关于 format 的描述，参见 sprintf()。                        args                                
返回值:
  返回值     根据 format （sprintf() 函数文档中有相关描述）参数指定的格式，在一个字符串中返回一系列值。   
示例:

<?php
print vsprintf("%04d-%02d-%02d", explode('-', '1988-8-1')); // 1988-08-01
?>
