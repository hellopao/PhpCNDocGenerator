语法:
   mixed fscanf    ( resource $handle   , string $format   [, mixed &$...  ] )
说明:
fscanf()函数和sscanf()相似，但是它从与handle关联的文件中接受输入并根据指定的format（定义于sprintf()的文档中）来解释输入。
参数:
  参数              handle           文件系统指针，是典型地由fopen() 创建的 resource(资源)。                  format                  参数格式是 sprintf() 文档中所描述的格式。                        ...                  The optional assigned values.                     
返回值:
  返回值     如果只给此函数传递了两个参数，解析后的值会被作为数组返回。否则，如果提供了可选参数，此函数将返回被赋值的数目。   可选参数必须用引用传递。   
示例:

<?php
$handle = fopen("users.txt", "r");
while ($userinfo = fscanf($handle, "%s\t%s\t%s\n")) {
    list ($name, $profession, $countrycode) = $userinfo;
    //... do something with the values
}
fclose($handle);
?>
