语法:
   array class_uses    ( mixed $class   [, bool $autoload = true  ] )
说明:
Thisfunctionreturnsanarraywiththenamesofthetraitsthatthegivenclassuses.Thisdoeshowevernotincludeanytraitsusedbyaparentclass.
参数:
  参数              class                  An object (class instance) or a string (class name).                        autoload                  Whether to allow this function to load the class automatically through       the __autoload() magic method.                     
返回值:
  返回值     An array on success, or FALSE on error.   
示例:

<?php
trait foo { }
class bar {
  use foo;
}
print_r(class_uses(new bar));
print_r(class_uses('bar'));
function __autoload($class_name) {
   require_once $class_name . '.php';
}
// use __autoload to load the 'not_loaded' class
print_r(class_uses('not_loaded', true));
?>
以上例程的输出类似于：
Array
(
    [foo] => foo
)
Array
(
    [foo] => foo
)
Array
(
    [trait_of_not_loaded] => trait_of_not_loaded
)
