语法:
   int gzpassthru    ( resource $zp   )
说明:
ReadstoEOFonthegivengz-filepointerfromthecurrentpositionandwritesthe(uncompressed)resultstostandardoutput.
参数:
  参数              zp                  The gz-file pointer. It must be valid, and must point to a file       successfully opened by gzopen().                     
返回值:
  返回值     The number of uncompressed characters read from gz   and passed through to the input, or FALSE on error.   
示例:

<?php
$fp = gzopen('file.gz', 'r');
gzpassthru($fp);
gzclose($fp);
?>
