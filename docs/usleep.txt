语法:
   void usleep    ( int $micro_seconds   )
说明:
以指定的微秒数延缓程序的执行。
参数:
  参数              micro_seconds                  暂停的时间以微秒计。1微秒（micro second）是百万分之一秒。                     
返回值:
  返回值     没有返回值。   
示例:

<?php
// Current time
echo date('h:i:s') . "\n";
// wait for 2 seconds
usleep(2000000);
// back!
echo date('h:i:s') . "\n";
?>
以上例程会输出：
11:13:28
11:13:30
