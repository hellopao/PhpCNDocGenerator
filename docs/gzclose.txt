语法:
   bool gzclose    ( resource $zp   )
说明:
Closesthegivengz-filepointer.
参数:
  参数              zp                  The gz-file pointer. It must be valid, and must point to a file       successfully opened by gzopen().                     
返回值:
  返回值     成功时返回 TRUE， 或者在失败时返回 FALSE。   
示例:

<?php
$gz = gzopen('somefile.gz','w9');
gzputs ($gz, 'I was added to somefile.gz');
gzclose($gz);
?>
