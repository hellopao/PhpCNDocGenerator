语法:
   string bcadd    ( string $left_operand   , string $right_operand   [, int $scale  ] )
说明:
Sumsleft_operandandright_operand.
参数:
  参数              left_operand                  The left operand, as a string.                        right_operand                  The right operand, as a string.                   scale此可选参数用于设置结果中小数点后的小数位数。也可通过使用bcscale() 来设置全局默认的小数位数，用于所有函数。      
返回值:
  返回值     The sum of the two operands, as a string.   
示例:

<?php
$a = '1.234';
$b = '5';
echo bcadd($a, $b);     // 6
echo bcadd($a, $b, 4);  // 6.2340
?>
