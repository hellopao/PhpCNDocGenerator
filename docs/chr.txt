语法:
   string chr    ( int $ascii   )
说明:
返回相对应于ascii所指定的单个字符。
参数:
  参数              ascii                  Ascii 码。                     
返回值:
  返回值     返回规定的字符。   
示例:

<?php
$str = "The string ends in escape: ";
$str .= chr(27); /* 在 $str 后边增加换码符 */
/* 通常这样更有用 */
$str = sprintf("The string ends in escape: %c", 27);
?>
