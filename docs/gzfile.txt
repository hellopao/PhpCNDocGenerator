语法:
   array gzfile    ( string $filename   [, int $use_include_path = 0  ] )
说明:
Thisfunctionisidenticaltoreadgzfile(),exceptthatitreturnsthefileinanarray.
参数:
  参数              filename                  The file name.                        use_include_path                  You can set this optional parameter to 1, if you       want to search for the file in the include_path too.                     
返回值:
  返回值     An array containing the file, one line per cell.   
示例:

<?php
$lines = gzfile('somefile.gz');
foreach ($lines as $line) {
    echo $line;
}
?>
