语法:
   array session_get_cookie_params    ( void   )
说明:
获取会话cookie的参数。
返回值:
  返回值     返回一个包含当前会话 cookie 信息的数组：                  &quot;lifetime&quot; - cookie 的生命周期，以秒为单位。                        &quot;path&quot; -  cookie 的访问路径。                        &quot;domain&quot; -  cookie 的域。                        &quot;secure&quot; -  仅在使用安全连接时发送 cookie。                        &quot;httponly&quot; - 只能通过 http 协议访问 cookie
