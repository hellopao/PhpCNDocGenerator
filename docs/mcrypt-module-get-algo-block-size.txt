语法:
   int mcrypt_module_get_algo_block_size    ( string $algorithm   [, string $lib_dir  ] )
说明:
返回指定算法的分组大小。
参数:
  参数              algorithm                  算法名称。                        lib_dir                  可选参数 lib_dir ，       表示在操作系统上包含模式模块的路径。                     
返回值:
  返回值     返回指定算法的分组大小，以字节为单位。
