语法:
      float sin       ( float $arg      )
说明:
sin()返回参数arg的正弦值。参数arg的单位为弧度。
参数:
  参数              arg                  单位是弧度的值。                     
返回值:
  返回值  arg 的正弦值   
示例:

<?php
// 返回值的精度由配置中的 precision 指示确定
echo sin(deg2rad(60));  //  0.866025403 ...
echo sin(60);           // -0.304810621 ...
?>
