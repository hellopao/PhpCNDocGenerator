语法:
   resource hash_init    ( string $algo   [, int $options = 0   [, string $key = NULL  ]] )
参数:
  参数              algo                  要使用的哈希算法名称，例如：&quot;md5&quot;，&quot;sha256&quot;，&quot;haval160,4&quot; 等。       如何获取受支持的算法清单，请参见 hash_algos()。                        options                  进行哈希运算的可选设置，目前仅支持一个选项：HASH_HMAC。       当指定此选项的时候，必须 指定 key 参数。                        key                  当 options 参数为 HASH_HMAC 时，       使用此参数传入进行 HMAC 哈希运算时的共享密钥。                     
返回值:
  返回值     返回哈希运算上下文资源，以供   hash_update()，   hash_update_stream()，hash_update_file(),   和 hash_final() 函数使用。   
示例:

<?php
$ctx = hash_init('md5');
hash_update($ctx, 'The quick brown fox ');
hash_update($ctx, 'jumped over the lazy dog.');
echo hash_final($ctx);
?>
以上例程会输出：
5c6ffbdd40d9556b73a21e63c3e0e904
